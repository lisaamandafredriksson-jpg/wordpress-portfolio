# wordpress-portfolio

Detta projekt är en personlig portfolio byggd som en del av min utbildning inom IT-utveckling. Syftet med uppgiften var att skapa en professionell, responsiv och användarvänlig webbplats med hjälp av WordPress som Content Management System (CMS).

## 🚀 Översikt
Webbplatsen fungerar som ett digitalt visitkort och portfolio för att visa upp mina färdigheter inom webbdesign, grafisk formgivning och innehållsskapande.

* **Roll:** Web Designer & CMS Administratör
* **Plattform:** WordPress
* **Status:** Slutfört skolprojekt

## 🛠 Verktyg och Tekniker
I projektet har jag arbetat med följande tekniker för att skapa designen och funktionaliteten:

* **Tema:** [Astra](https://wpastra.com/) – Ett lättviktigt och anpassningsbart tema för hög prestanda.
* **Sidbyggare:** [Elementor](https://elementor.com/) – Använt för att bygga unika layouter och anpassade sektioner.
* **Design:** Fullt responsiv layout anpassad för desktop, surfplatta och mobil.

## 📦 Installerade & Konfigurerade Plugins
För att utöka sajtens funktionalitet har jag installerat och konfigurerat följande verktyg:

* **Contact Form 7 / WPForms Lite:** För hantering av sajtens kontaktformulär.
* **Ultimate Addons for Elementor:** För avancerade designelement och widgets.
* **Wordfence Security:** För att implementera grundläggande brandvägg och säkerhetsskanning.
* **My Sticky Bar:** För en låst navigationsmeny som förbättrar användarupplevelsen.
* **Starter Templates:** Använts som grund för att sedan anpassas till ett personligt varumärke.
* **All-in-One WP Migration:** Använt för export och säkerhetskopiering av hela projektet.

## 💡 Lärdomar
Genom detta projekt har jag fått praktisk erfarenhet av:
* Informationsarkitektur och planering av användarflöden (UX).
* Grafisk anpassning av färger, typografi och bildspråk för webben.
* Installation, uppdatering och felsökning av plugins och teman.
* Grundläggande säkerhetstänk kring WordPress-installationer.

## 📥 Installation (Lokal miljö)
För att se eller vidareutveckla projektet lokalt:

1.  Sätt upp en lokal WordPress-instans (t.ex. med [LocalWP](https://localwp.com/)).
2.  Installera och aktivera pluginet **All-in-One WP Migration**.
3.  Importera `.wpress`-filen som finns i detta repository.
